== WP Simple Feedback ==

WP Simple Feedback is a plugin for WordPress. With this plugin you are able to let your registered users vote for serveral topics.

=== Installation ===

1. Copy the folder to /wp-content/plugins/ and activate the plugin in the WordPress Admin Panel.
2. Create two pages where the topics should be shown.
3. Add the shortcode [topics-open] to the page where the topics appear which are currently in vote.
4. Add the shortcode [topics-closed] to the page where the topics appear which have been voted.
5. Head to "Topics" -> "Settings" to set the needed redirection and the expiration time.
6. Head to "Topics" to add topics.
